#### 购车价格、税费以及保险
* `指导价`：汽车厂商根据生产成本和预估市场确定的销售价格。受汽车`厂商优惠`影响
* `裸车价`：4s店根据实际市场确定的销售价格，受`4s店优惠`和`厂商优惠`的影响
* `落地价`：可以合法上路的最终成交价格，包括`裸车价`+`购置税`+`保险`+`牌照`，受`购置税减免`等税收优惠政策的影响
